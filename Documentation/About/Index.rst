=====
About
=====

The users country and preferred language and other facts will be detected. Depending on configurable rules the user can be
redirected to other languages or pages. It is also possible to deny access to configurable pages for configurable countries.

.. _about-compatibility:

Compatibility
=============

We are currently supporting following TYPO3 versions:

.. csv-table:: Version Matrix
   :header: "Extension Version", "TYPO3 v13 Support", "TYPO3 v12 Support"
   :align: center

        "13.x", "yes", "no"
        "12.x", "no", "yes"

.. _about-compatibility-outdatedVersions:

Outdated Versions
-----------------
For all other versions no more free bug fixes and new features will be provided by the authors.

.. _about-ip-database:

IP Database
===========

This site or product includes IP2Location LITE data available from `https://lite.ip2location.com <https://lite.ip2location.com>`__.
